## Заметки для выбора

1. **Мобильные паттерны:** `Sheet` / `Drawer` вместо широких `Dialog`; большие тач-таргеты.
2. **Критичные действия:** `Alert Dialog`, не обычный `Dialog`.
3. **Таблицы с бизнес-логикой:** `Data Table`, не «голый» `Table`.
4. **Палитра команд:** `Command` отлично сочетается с глобальным hotkey — держите список действий коротким.
5. **Доступность:** компоненты на Radix; кастомная разметка не должна ломать фокус и `aria-`*.

Источник списка компонентов: [ui.shadcn.com/docs/components](https://ui.shadcn.com/docs/components).
